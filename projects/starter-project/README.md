# Starter Project

새 프로젝트 시작용 템플릿입니다.

## Structure

- `python/`: Python 앱
- `web/`: npm 기반 웹 앱

## Python

```bash
cd python
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
python src/main.py
```

## Web (npm)

```bash
cd web
npm install
npm run dev
```

기본 dev 서버는 `http://localhost:5173` 입니다.
