## Project Overview

This is a Python project currently in early setup phase with minimal structure.

## Development Environment

The project uses a virtual environment located in `.venv/`.

To activate the virtual environment:
```bash
# Windows
.venv\Scripts\activate

# Linux/macOS
source .venv/bin/activate
```

## Running the Code

```bash
python main.py
```

## Current Structure

- `main.py` - Main entry point (currently contains PyCharm template code)
- `.venv/` - Python virtual environment
